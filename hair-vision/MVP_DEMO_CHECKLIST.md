## ⚠️ 常见问题与解决方案

### 问题1: API Key 未配置
**症状**: 生成失败，错误提示"Gemini API key not configured"
**解决**: 
```bash
# 在 hair-vision/.env.local 添加：
GEMINI_API_KEY=your_key_here
# 重启开发服务器
```

### 问题2: 生成时间过长
**症状**: 等待超过1分钟仍未生成
**解决**: 
- 检查网络连接
- 检查API配额是否用尽
- 使用重试功能
- 降级到Flash模型（自动）

### 问题3: 照片质量问题
**症状**: 生成效果不理想
**解决**: 
- 确保光线充足
- 面部清晰可见
- 使用正面照片（单张模式）
- 或使用多角度模式（V1.5）

### 问题4: 浏览器兼容性
**症状**: 相机无法打开
**解决**: 
- 使用Chrome/Safari最新版本
- 确保已授权摄像头权限
- 可以改用"从相册选择"功能
