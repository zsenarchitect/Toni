# 市场研究工具使用指南

## 概述

市场研究工具用于验证演示文稿中基于假设的数据，确保所有数据都有可靠来源支持。

## 功能

### 1. 自动验证工具

#### NYC 沙龙数量验证
- **路径**: `/admin/research`
- **功能**: 使用 Google Places 和 Yelp API 自动查询 NYC 地区的沙龙数量
- **使用方法**: 点击"验证 NYC 沙龙数量"按钮

### 2. 手动添加研究数据

#### API 端点
- **POST** `/api/research/data` - 添加研究数据
- **GET** `/api/research/data` - 获取研究数据
- **PATCH** `/api/research/data` - 更新验证状态

#### 示例请求
```json
POST /api/research/data
{
  "category": "market_size",
  "title": "全球美发市场规模",
  "value": "$105B (2024)",
  "source": "IBISWorld Market Research",
  "sourceUrl": "https://example.com/report",
  "verified": true,
  "notes": "基于行业报告"
}
```

### 3. 数据分类

- **market_size**: 市场规模数据
- **customer_satisfaction**: 客户满意度数据
- **competition**: 竞争分析数据
- **pricing**: 定价数据
- **local_market**: 本地市场数据

## 需要验证的关键假设

### 高优先级（影响业务决策）

1. **73% 客户不满意率**
   - 需要来源：行业调查、学术研究、客户反馈分析
   - 验证方法：客户访谈、在线调查、第三方研究报告

2. **市场规模数据**
   - 全球美发市场: $105-108B (2024)
   - 美国美发市场: $20.12B (2025)
   - 高端美发市场: $8.13B (2024) → $12.62B (2033)
   - 验证方法：行业报告（IBISWorld, Statista, Grand View Research）

3. **NYC 沙龙数量**
   - 总沙龙数: 8,000-10,000
   - 高端沙龙 ($150+): 500-800
   - 验证方法：使用 `/admin/research` 工具自动验证

4. **定价接受度**
   - $199-499/月 价格接受度
   - 验证方法：客户访谈、竞品分析、试点测试

### 中优先级（影响运营策略）

5. **颜色修正成本**
   - $300+ 平均成本
   - 验证方法：行业调查、沙龙访谈

6. **等待时间**
   - 6个月等待修复时间
   - 验证方法：客户访谈、行业标准

7. **升级转化率**
   - 15-20% 升级转化率
   - 验证方法：试点客户数据收集

## 使用流程

1. **识别假设**: 在演示文稿中标记所有需要验证的数据
2. **收集数据**: 
   - 使用自动工具（如 NYC 沙龙数量）
   - 手动添加研究数据（通过 API 或管理界面）
3. **验证数据**: 标记数据为已验证，并添加来源
4. **更新演示文稿**: 根据验证结果更新演示文稿内容

## 最佳实践

1. **始终标注来源**: 每个数据点都应该有明确的来源
2. **区分已验证和未验证**: 在演示文稿中明确标记数据状态
3. **定期更新**: 市场数据会变化，定期重新验证
4. **记录验证过程**: 在 notes 字段中记录验证方法和过程
